# aish — AI Shell

`aish` is the AI-OS native shell. It is POSIX-compatible with enhancements
for usability: tab completion, history, colored prompts, and AI integration.

## Building

```sh
make
make install  # installs to /bin/aish
```

## Features

- POSIX sh compatible
- Pipes and redirections
- Job control (fg/bg/&)
- Tab completion
- Command history (up/down arrows)
- Customizable PS1/PS2 prompts
- Built-in help system
- Color support
