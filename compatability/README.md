# Compatibility Layers

This directory contains integration layers for foreign software ecosystems:

| Directory | Purpose |
|-----------|---------|
| `packages/` | Universal Package Compatibility Layer (UPCL) |
| `android/`  | Waydroid + microG Android support |
| `windows/`  | Wine + Proton Windows compatibility |
| `vm/`       | VM wrappers (Windows Legacy, Banking, Dev, etc.) |
| `nvidia/`   | NVIDIA driver integration |
| `gpu/`      | AMD/Intel GPU setup |
| `btrfs/`    | Btrfs snapshot/rollback integration |
