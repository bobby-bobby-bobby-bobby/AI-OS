# VM Wrappers

Specialized virtual machine wrappers for AI-OS.

| Script | Purpose |
|--------|---------|
| `vm-manager.sh` | Central VM management CLI |
| `windows-legacy.sh` | Windows Legacy Mode VM (QEMU) |
| `banking-vm.sh` | Hardened Banking VM |
| `suspicious-vm.sh` | Disposable sandbox for suspicious files |
| `dev-vm.sh` | Developer VM |
| `browser-vm.sh` | Browser isolation VM |
| `templates/` | QEMU configuration templates |
