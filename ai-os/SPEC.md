# AI-Generated Linux OS Specification

This repository contains a Linux distribution where all userland components,
init system, package manager, installer, configuration files, and documentation
are generated by AI models inside VS Code.

## Non-AI Components (required for hardware support)
- Linux kernel (fetched externally)
- musl libc
- tcc or clang toolchain
- busybox (optional fallback)
- NVIDIA proprietary driver support
- QEMU for legacy mode VM

## AI-Generated Components
- init system
- shell
- core utilities
- package manager
- service manager
- installer
- configuration system
- branding, themes, docs
- Windows legacy mode VM wrapper

## Goals
- Fully bootable on real hardware
- NVIDIA GPU support
- Legacy Windows software support via VM
- Minimal, clean, modular architecture
- AI-maintained and AI-evolved
