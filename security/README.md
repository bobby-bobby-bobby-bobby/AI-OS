# Security

AI-OS security integration using established open-source tools.

| File/Dir | Purpose |
|----------|---------|
| `apparmor/` | AppArmor profiles for system applications |
| `firejail/` | Firejail sandbox profiles |
| `bubblewrap/` | Bubblewrap sandbox wrappers |
| `firewall.sh` | Firewall configuration (nftables/iptables) |
| `adblock/` | DNS-level ad/tracker blocking |
| `encrypted-home.sh` | LUKS/fscrypt encrypted home setup |
| `auto-update.sh` | Automatic security updates |
| `uninstall.sh` | Universal uninstaller |
| `browser-isolate.sh` | Browser isolation wrapper |
